## 🦌 Introdução

Este trabalho tem como implementação um simulador de propagação de incêndio em uma floresta, considerando também, a existência de um animal dentro da floresta que se movimenta a cada iteração, assim como as chamas do incêndio que se propagam de maneira ortogonal a cada iteração.
